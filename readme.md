# sung

> The [Sung Kim ](https://sungkim.co/) CLI

## Usage

Install Node.js, then:

```
$ npx sung
```

## Environment

_Currently only supported on [iTerm >=3](https://www.iterm2.com/downloads.html)._
due to dependency on [term-img](https://www.npmjs.com/package/term-img).
